🎬 NEP_FLIX
A Netflix-inspired streaming web application built using HTML, CSS, and JavaScript.
Designed with a modern UI, cinematic effects, and structured file management.

🚀 Features
🎥 Intro video loading screen
🎨 Cinematic hero section
🎞️ Dynamic movie cards
▶️ Watch page with custom UI controls
🔐 Signup page
📱 Responsive layout
✨ Smooth animations and transitions

📂 Project Structure

NEP_FLIX/
│
├── assets/                # Static assets
│   ├── img/               # Images
│   ├── posters/           # Movie posters
│   └── videos/            # Video files
│
├── home_html.html         # Homepage HTML
├── home_style.css         # Homepage styling
├── home_style.js          # Homepage scripts
│
├── nepflix-watch.html     # Watch page HTML
├── nepflix-watchcss.css   # Watch page styling
├── nepflix-watchjs.js     # Watch page functionality
├── nepflix-watchui.js     # Watch page UI logic
│
├── open.html              # Opening/landing page
├── open_style.css         # Opening page styling
│
├── signup_html.html       # Signup page HTML
└── signup_style.css       # Signup page styling
🛠️ Technologies Used

HTML5 – Structure
CSS3 – Styling & animations
JavaScript (Vanilla JS) – Functionality & UI interactions

🎯 How to Run the Project
Download or clone the repository:
git clone https://github.com/ayushshah-xo/NEP_FLIX.git
Open open.html or home_html.html in your browser.
No additional dependencies required.
